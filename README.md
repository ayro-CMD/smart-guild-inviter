# Smart Guild Inviter

A WoW addon that lets you invite people to your guild with one click. Works on EVERY version (Classic, TBC, Wrath, Retail, you name it).

## What it does

- Adds a movable "Invite to Guild" button
- Auto-checks if target is a player, has the right level, isn't already in a guild
- Optional auto-whisper when you invite someone
- Minimap icon to show/hide the button

## How to install

1. Download the `SmartGuildInviter` folder
2. Drop it into `World of Warcraft\_retail_\Interface\AddOns\` (or whatever version you're playing)
3. Restart WoW or type `/reload`
4. Look for the circle icon on your minimap and click it
5. Target a player and click "Invite to Guild"
